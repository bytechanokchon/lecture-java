## ก่อกำเนิดภาษาใหม่จากแนวคิดของ Java (The C# Connection)
แนวคิดที่เกิดขึ้นจากภาษา Java กลายเป็นลากฐานการออกแบบสำหรับภาษาใหม่ ๆ ที่กำลังจะเกิดขึ้นบนโลกใบนี้ เพราะความสำเร็จของภาษานี้ ไม่สามารถมองข้ามไปได้

ตัวอย่างเช่น การเกิดขึ้นของภาษา C# ซึ่งถูกพัฒนโดย Microsoft เพื่อรองรับการเกิดขึ้นของ .NET Framework โดยทั้ง C# และ Java มีหลายอย่างที่คล้ายกันมาก ไม่ว่าจะเป็น ไวยกรณ์ การเขียนโปรแกรมเชิงวัตถุ การใช้งานวัตถุที่เหมือนกัน
