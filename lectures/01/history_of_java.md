# Histoty Of Java
**ภาษาจาวา เกิดขึ้นได้อย่างไร และเพราะอะไรทำให้มันมีความสำคัญ**

โดยจุดเริ่มต้นของ java เกิดขึ้นมาจาก 2 เหตุผลหลัก คือ

1. เพื่อปรับตัวให้เข้ากับสภาพแวดล้อมและการใช้งานที่เปลี่ยนแปรงไป
2. เพื่อปรับปรุงและพัฒนาศิลปะของการเขียนโปรแกรม

โดยการพัฒนาภาษานี้ จะได้รับแรงผลักดันจาก 2 เหตุผลนี้เป็นหลัก

## Java’s Lineage
ภาษา Java มีความเกี่ยวข้องกับ C++ ซึ่งถูกสืบทอดมาจาก C โดยตรง ทำให้คุณสมบัติส่วนใหญ่ของ Java จะถูกสืบทอดมาจากภาษาทั้ง 2 นี้ โดยตัวภาษาได้รับคุณสมบัติเกี่ยวกับไวยากรณ์มาจากภาษา C และคุณสมบัติเกี่ยวกับการเขียนโปรแกรมเชิงวัตถุมาจาก C++

โดยตลอดระยะเวลาที่ผ่านมา ตัวภาษาถูกปรับปรุงมาเรื่อย ๆ (จนปัจจุบันเวอร์ชัน 21) เพื่อแก้ไขปัญหาพื้นฐานของภษาโปรแกรมรุ่นก่อน ๆ 

## The Birth of Modern Programming: C
เมื่อภาษา C เกิดขึ้น มันเปลี่ยนแปรงโลกของการเขียนโปรแกรมไปตลอดการ ทั้งวิธีคิด ประสิทธิภาพ โครงสร้าง ที่่สามารถมาแทนที่ assembly ได้เลยในการสร้างโปรแกรม

โดยเมื่อมีการออกแบบภาษาคอมพิวเตอร์ใหม่ ๆ มา มักจะคำถึงถึงสิ่งต่าง ๆ ดังต่อไปนี้
1. ความง่ายในการใช้งาน เทียบกับพลังงาน (ease-of-use versus power)
2. ความปลอดภัยเทียบกับประสิทธืภาพ (safety versus efficiency)
3. ความแข็งแกร่งเทียบกับความยิเหยุ่น (rigidity versus extensibility)

ก่อนที่จะมีภาษา C โปรแกรมเมอร์มักจะต้องเลือกว่าจะเอาภาษาที่มีคุณลักษณะชุดใดชุดหนึ่งเป็นพิเศษ เช่น
- Fortran ใช้สำหรับเขียนโปรแกรมที่มีประสิทธิภาพพอสมควรได้สำหรับแอปพลิเคชันทางวิทยาศาสตร์ แต่ก็ไม่เหมาะกับการนำมาเขียนระบบขนาดใหญ่
- Basic เรียนรู้ได้ง่าย แต่ไม่ทรงพลัง ขาดโครงสร้างที่่ทำให้การพัฒนาโปรแกรมขนาดใหญ่สามารถทำได้ยาก
- Assembly สามารถใช้งานพัฒนาโปรแกรมประสิทธิภาพสูงได้ แต่ไม่ง่ายที่จะเรียนรู้และใช้งาน และดีบักโค้ดได้ยาก

และภาษาในยุคก่อน ไม่ได้ถูกออกแบบมาสหรับการเขียนโปรแกรมเชิงวัตถุ แต่จะใช้หลักการ GOTO แทน ทำให้มักจะเกิดปัญหา "โค้ดสปาเก็ตตี้" ซึ่งเป็นกลุ่มของคำสั่งที่กระโดดไปมา และมีเงื่อนไขพันกันยุ่งเหยิง

## C++: The Next Step
ตลอดเวลาที่ผ่านมา ภาษา C ได้รับความนิยม และถูกใช้อย่างแพร่หลายจนถึงปัจจุบัน แต่ทำไมต้องมีภาษาโปรแกรมอื่นเกิดขึ้นมาละ ?

ปัญหาของภาษา C คือ มีความซับซ้อน ยิ่งโปรแกรมใหญ่ ก็ยิ่งมีความซับซ้อน เกินความที่โปรแกรมเมอร์จะรับไหว

เพื่อแก้ปัญหาดังกล่าว จึงได้เกิดภาษา C++ ขึ้น และได้มีการผลักดันโครงสร้างใหม่ขึ้นมา คือ การเขียนโปรแกรมเชิงวัตถุ (Object-Oriented Programming)
