## คำศัพท์ยอดนิยมของ Java (The Java Buzzwords)
นอกจากความสามารถในการพกพาแล้ว ยังมีสิ่งอื่น ๆ ที่เราควรรู้ไว้ด้วย โดยมันจะเป็นคำศัพท์เฉพาะทาง ดังนี้

1. simple
2. secure
3. portable
4. object-oriented
5. robust
6. multithreaded
7. archtecture-neutral
8. interpreted
9. high performance
10. distributed
11. dynamic

### simple
ภาษาถูกออกแบบมาให้เรียนรู้และใช้งานได้ง่าย

### object-oriented
โดยแนวคิด "ทุกอย่าเป็นออบเจ็กต์" ทำให้ชุดคำสั่งของ Java นั้นสะอาดตา ใช้งานง่าย และไปเป็นตามหลักการ สามารถขยายความสามารถได้

### robust
Java เป็นภาษาที่กำหนดประเภทของข้อมูลอย่างเข้มงวด และช่วยตรวจสอบข้อผิดพลาดตั้งแต่ขั้นตอนการคอมไพล์ ทำให้เป็นเรื่องยากมากที่จะเกิดข้อผิดพลาดที่ไม่คาดคิดหรือไม่สามารถเดาได้ระหว่างการใช้งานจริง ทำให้ได้รับความน่าเชื่อถือ

โดยสาเหตุหลัก 2 ประการที่ทำให้โปรแกรมล้มเหลว คือ
1. ข้อผิดพลาดในการจัดการหน่วยความจำ
2. การจัดการสภาวะผิดปกติที่ไม่ถูกต้อง (ข้อผิดพลาดเกิดขึ้นขณะรันไทม์ เช่น ไม่พบไฟล์)

### multithreaded
Java รองรับการเขียนโปรแกรมแบบมัลติเธรด ซึ่งทำให้เราสามารถทำงานหลายอย่างพร้อมกันได้

### archtecture-neutral
การเกิดขึ้นของ Java Virtual Machine ทำให้เราสามารถรับประกันได้ว่า "เขียนครั้งเดียว ทำงานได้ทุกที่ ทุกเวลา ตลอดไป"

ไม่เหมือนกับโปรแกรมสมัยก่อน ที่เราอาจแค่เปลี่ยน ram ก็สามารถทำให้โปรแกรมไม่สามารถทำงานได้อีกได้

### high performance
เป็นภาษาที่มีประสิทธิภาพสูง แม้ว่าจะเป็น native-code ก็ตาม แต่ด้วยการออกแบบที่พิถีพิถัน ทำให้ได้ประสิทธิภาพที่ใกล้เคียงกับ machine code

### distributed
ตัวภาษาถูกออกแบบมาสำหรับสภาพแวดล้อมแบบกระจายศูนย์ ด้วยความสามารถจัดการกับโปรโตคอล TCP/IP ได้ และยังมีความสามารถการเรียกใช้งานเมธอดจากระยะไกล (Remote method invocation หรือ RMI)

### dynamic
ใน Java จะเก้บข้อมูลชนิดของข้อมูลไว้จำนวนมากในช่วงรันไทม์เพื่อให้ JVM สามารถตรวจสอบความถูกต้อง ตัดสินการเรียกใช้งานแบบไดนามิก และการเชื่อมโยงโค้ดใหม่อย่างปลอดภัย ซึ่งทำให้ Java มีความทนทานสูง และรองรับการอัปเดตโค้ดระหว่างการทำงานของระบบ
