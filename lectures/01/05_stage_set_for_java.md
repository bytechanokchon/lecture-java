## เวทีพร้อมแล้วสำหรับภาษา Java! (The Stage Is Set for Java)
ในช่วงปลายปี 1980 จนถึงต้นปี 1990 การเขียนโปรแกรมด้วยภาษา C++ ได้รับความนิยมเป็นอย่างมาก
ด้วยความสามารถของการเขียนโปรแกรมเชิงวัตถุ ที่ทำให้สามารถสร้างโปรแกรมได้ทุกรูปแบบ 

แต่เมื่อเวลาผ่านไป เทคโนโลยีใหม่ก็เข้ามา โดยเทคโนโลยีใหม่ที่เข้ามาคือ world wide web 
ซึ่งปฏิวัติอุสาหกรรมของการเขียนโปรแกรมคอมพิวเตอร์ไปอีกขั้น
